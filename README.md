# 📝 Software Persona - Todo App (Staj Projesi)

Bu proje, modern web teknolojileri kullanılarak geliştirilmiş, **CRUD** işlemleri yapabilen ve **API** entegrasyonu içeren bir Görev Yönetimi uygulamasıdır.

## 🚀 Özellikler
- **CRUD İşlemleri:** Görev ekleme, listeleme, güncelleme ve silme.
- **API Entegrasyonu:** İlk veriler Axios ile JSONPlaceholder üzerinden çekilir.
- **Yerel Depolama:** Veriler `LocalStorage` ile tarayıcıda saklanır (Sayfa yenilendiğinde kaybolmaz).
- **Modern Tasarım:** Tailwind CSS v4 ve Lucide-React ikonları ile geliştirildi.
- **Responsive:** Mobil ve masaüstü cihazlarla tam uyumlu.

## 🛠️ Kullanılan Teknolojiler
- **Framework:** React 18
- **Styling:** Tailwind CSS v4
- **HTTP Client:** Axios
- **Icons:** Lucide-React
- **Build Tool:** Vite

## 📦 Kurulum
Projeyi yerelde çalıştırmak için:
1. `git clone https://github.com/furkancelik1/software-persona-todo.git`
2. `npm install`
3. `npm run dev`
